## 11) If Shortcuts Are Not Working

If a shortcut is not working:

1. Open **Settings** with `Ctrl+Alt+S`
2. Go to **Keymap**
3. Search for the action name
4. Check whether your current keymap is:
   - IntelliJ IDEA
   - VS Code
   - Eclipse
   - Other custom mapping

Because shortcuts can change if you are not using the default IntelliJ keymap.

## 12) Quick Trainer Tip

For Java developers, the most useful shortcut groups are:

- **Coding** тЖТ `Ctrl+Space`, `Alt+Enter`, `Alt+Insert`
- **Navigation** тЖТ `Ctrl+B`, `Alt+F7`, `Ctrl+N`
- **Refactoring** тЖТ `Shift+F6`, `Ctrl+Alt+Shift+T`
- **Debugging** тЖТ `Shift+F9`, `F8`, `F7`, `Shift+F8`
- **Git** тЖТ `Ctrl+K`, `Ctrl+Shift+K`

If you master these, your speed in IntelliJ will improve a lot.
